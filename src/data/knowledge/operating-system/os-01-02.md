# 第一章 操作系统概述 (1.2) —— 操作系统的发展与分类

## 1. 发展阶段
1. **单道批处理系统**:
    - **优点**: 缓解了人机矛盾（手工操作快、人工慢）。
    - **缺点**: 内存中仅有一道作业，CPU 利用率低。
2. **多道批处理系统 (OS 开始出现的标志)**:
    - **原理**: 内存中存放多道相互独立的程序，当某道程序进行 I/O 时，CPU 切换至另一道程序。
    - **优点**: 资源利用率高、系统吞吐量大。
    - **缺点**: 无交互能力，作业平均周转时间长。
3. **分时操作系统**:
    - **核心**: 以时间片为单位，将 CPU 轮流分配给各联机终端用户。
    - **特征**: 多路性、交互性、独立性、及时性。
4. **实时操作系统**:
    - **硬实时系统**: 必须在规定时间内完成（如导弹控制）。
    - **软实时系统**: 可以忍受一定的延时（如订票系统）。
