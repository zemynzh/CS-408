# 第四章 文件管理 (4.3) —— 磁盘管理

## 1. 磁盘结构 (Disk Structure)
- **盘面、柱面、磁道、扇区**。
- **寻址方式**: `(柱面号, 盘面号, 扇区号)`。
- **磁盘读写时间**:
    - $T = T_{seek} + T_{rot} + T_{transfer}$
    - **寻道时间 ($T_{seek}$)**: 移动磁臂到指定磁道。
    - **延迟时间 ($T_{rot}$)**: 旋转盘面到指定扇区。$T_{rot} \approx \frac{1}{2r}$ ($r$ 为转速)。
    - **传输时间 ($T_{transfer}$)**: 数据传输出去。

## 2. 磁盘调度算法 (Disk Scheduling)
1. **先来先服务 (FCFS)**: 简单，公平，但寻道距离长。
2. **最短寻找时间优先 (SSTF)**:
    - **原理**: 选择与当前磁头最近的磁道。
    - **缺点**: 易导致 **饥饿**（偏向靠近磁头的请求）。
3. **扫描算法 (SCAN - 电梯调度)**:
    - **原理**: 磁头从一端向另一端移动，中途处理所有请求，直到到达 **最边缘** 才掉头。
    - **优点**: 性能好，无饥饿。
4. **循环扫描 (C-SCAN)**:
    - **原理**: 磁头只在单向移动时处理请求，到达边缘后 **立即返回** 另一端。
    - **优点**: 请求等待时间更均匀。
5. **LOOK 与 C-LOOK**:
    - **改进**: 磁头不必到达物理边缘，只需到达 **最边缘的请求** 即可掉头/返回。

## 3. 磁盘初始化
- **物理格式化 (低级格式化)**: 划分扇区。
- **分区**: 逻辑磁盘划分。
- **逻辑格式化**: 建立文件系统（根目录、FAT）。
