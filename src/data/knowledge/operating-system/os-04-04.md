# 第四章 文件管理 (4.4) —— 文件系统的结构和保护

## 1. 文件系统的层次结构
- **用户接口**
- **逻辑文件系统**: 负责文件结构、目录。
- **基本文件系统**: 负责块寻址。
- **I/O 控制层**: 负责物理读写。

## 2. 文件的保护
1. **口令保护**: 设置密码，检查匹配。
2. **存取控制 (Access Control)**: 
    - **存取控制表 (ACL)**: 列表记录各用户对该文件的权限（读、写、执行）。
    - **权限标志 (rwx)**: UNIX 系统采用，分为 `拥有者 | 组 | 其他人`。
    - **优点**: 效率高。
3. **加密技术**: 将文件内容进行加密处理。

## 3. 文件的存取
1. **顺序存取**: 
    - 磁带、顺序文件。
    - 按顺序访问记录。
2. **随机存取 (直接存取)**: 
    - 磁盘、索引文件。
    - 直接访问某一块（通过地址计算）。

## 4. 磁盘碎片
- **内部碎片**: 块内未利用空间。
- **外部碎片**: 块与块之间的碎片（可通过紧凑技术解决）。
