# 差错控制

## 1. 检错编码
### 奇偶校验码
- 奇校验：1 的个数为奇数。
- 偶校验：1 的个数为偶数。
- 局限：只能检测奇数位错误，无法纠错。

### 循环冗余校验 (CRC)
- **原理**：基于模2除法。
- **步骤**：
  1. 给定生成多项式 $G(X)$（对应二进制串）。
  2. 若 $G(X)$ 阶数为 $r$，在数据 $M$ 后补 $r$ 个 0。
  3. 用补 0 后的 $M$ 除以 $G(X)$，余数即为 **FCS** (帧检验序列)。
- **注意**：数据链路层只提供“无差错接收”，即不接受有错帧。但并不意味着“可靠传输”。

## 2. 纠错编码
### 海明码 (Hamming Code)
- **原理**：$L-1 = D + S$ 或更常用的公式 $2^k \ge n + k + 1$。
- **纠错能力**：可以发现两位错，纠正一位错。
