# IP 地址与子网划分 (CIDR)

## 1. IP 地址的分类
- A 类：0.0.0.0 ~ 127.255.255.255
- B 类：128.0.0.0 ~ 191.255.255.255
- C 类：192.0.0.0 ~ 223.255.255.255
- D 类：224.0.0.0 ~ 239.255.255.255（组播）
- E 类：240.0.0.0 ~ 255.255.255.255（保留）

## 2. 子网划分
- **子网掩码**：1 表示网络号/子网号，0 表示主机号。
- **子网地址** = IP 地址 AND 子网掩码。

## 3. CIDR (无分类编址)
- **形式**：IP 地址 / 网络前缀长度。
- **路由聚合 (Supernetting)**：通过更短的前缀掩盖多个小网段。
- **最长前缀匹配**：查找转发表时，选择匹配位数最多的路由。

## 4. 特殊 IP 地址
- 全 0：本主机。
- 全 1：广播。
- 127.0.0.1：回环。
- **私有地址**：10.x, 172.16.x~31.x, 192.168.x。
- **NAT (网络地址转换)**：转换私网 IP 与公网 IP。
- **DHCP (动态主机配置协议)**：应用层协议，基于 UDP。
