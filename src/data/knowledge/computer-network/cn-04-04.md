# ICMP 与 ARP

## 1. ARP 协议 (Address Resolution Protocol)
- **功能**：由 IP 地址获取 MAC 地址。
- **原理**：
  1. 查 ARP 缓存。
  2. 未命中则广播 ARP 请求，目的主机单播 ARP 响应。
  3. 更新本地 ARP 缓存。
- **层级**：介于链路层和网络层之间，通常归为网络层。

## 2. ICMP 协议 (Internet Control Message Protocol)
- **功能**：报告差错、查询状态。
- **两类报文**：
  1. **差错报告报文**：终点不可达、源点抑制、超时、参数问题、改变路由。
  2. **询问报文**：回送请求（Ping）、时间戳请求。
- **应用**：
  - `Ping`：基于 ICMP 回送请求报文。
  - `Traceroute`：利用 TTL 和 ICMP 时间超过报文。
- **封装**：ICMP 封装在 IP 数据报中。
