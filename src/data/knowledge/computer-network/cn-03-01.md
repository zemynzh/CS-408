# 数据链路层功能与成帧

## 1. 数据链路层的功能
1. **为网络层提供服务**：无确认无连接、有确认无连接、有确认面向连接。
2. **链路管理**：连接建立、维持、释放。
3. **帧同步与成帧**：封装成帧（Encapsulation）。
4. **透明传输**：无论数据中出现什么比特组合，都能正确传输。
5. **差错控制**：检错编码、纠错编码。
6. **流量控制**：限制发送方发送速率。

## 2. 封装成帧的方法
- **字符计数法**：帧首用一个字节表示帧长度（含首部）。
- **字符填充法**：使用 SOH 和 EOT 作为帧起始和结束。数据中的 SOH/EOT 前加 ESC。
- **零比特填充法 (0-bit Insertion)**：帧首尾为 `01111110` (6个1)。数据中若有 5 个连续的 1，自动填充一个 0。
- **违规编码法**：曼彻斯特编码中不用的电平组合作为边界。

## 3. MTU
最大传输单元 (Maximum Transmission Unit)。以太网 MTU 通常为 1500 字节。
