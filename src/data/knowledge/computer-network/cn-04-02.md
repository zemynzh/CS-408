# IPv4 协议与分片

## 1. IPv4 报文首部
- **固定部分**：20 字节。
- **版本**：4 (IPv4)。
- **首部长度**：单位为 4 字节。最大 60 字节。
- **区分服务**：TOS。
- **总长度**：单位为 1 字节。首部 + 数据。
- **生存时间 (TTL)**：跳数，每过一个路由器减 1，为 0 则丢弃。
- **协议**：TCP (6), UDP (17), ICMP (1)。
- **首部校验和**：仅校验首部。

## 2. IP 分片
- **原因**：IP 数据报超过链路层 MTU。
- **关键字段**：
  - **标识 (Identification)**：同一数据报的分片标识相同。
  - **标志 (Flags)**：
    - DF=1 (Don't Fragment)：不可分片。
    - MF=1 (More Fragments)：后面还有分片。
  - **片偏移 (Offset)**：单位为 **8 字节**。当前分片在原数据报中的位置。
- **重组**：在目的主机端重组。
