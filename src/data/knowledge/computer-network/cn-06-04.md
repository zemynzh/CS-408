# 电子邮件

## 1. 组成部分
- 用户代理 (UA)、邮件服务器、邮件协议。

## 2. SMTP 协议 (Simple Mail Transfer Protocol)
- **功能**：发送邮件。
- **特点**：基于 TCP (25端口)，推送协议 (Push)。只能发送 ASCII 文本。
- **MIME**：多用途因特网邮件扩展，允许发送二进制（附件）。

## 3. POP3 / IMAP (读取协议)
- **POP3**：拉取协议 (Pull)，离线处理。默认下载后在服务器删除。
- **IMAP**：在线处理，同步邮件状态。
- **HTTP (Webmail)**：用户与邮件服务器之间使用 HTTP。
