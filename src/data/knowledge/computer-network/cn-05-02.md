# TCP 特点与报文

## 1. TCP 协议 (Transmission Control Protocol)
- **特点**：面向连接、可靠交付、全双工、面向字节流。
- **套接字 (Socket)**：IP 地址 + 端口号。

## 2. TCP 报文段首部
- **固定部分**：20 字节。
- **序号 (Seq)**：本报文段数据的第一个字节的序号。
- **确认号 (Ack)**：期望收到对方下一个报文段数据的第一个字节。
- **首部长度 (Offset)**：单位为 4 字节。
- **标志位**：
  - **URG**：紧急位。
  - **ACK**：确认位。ACK=1 时确认号有效。
  - **PSH**：推送位。
  - **RST**：复位位。连接出错。
  - **SYN**：同步位。连接建立。
  - **FIN**：终止位。连接释放。
- **窗口 (Window)**：接收方允许发送方发送的数据量。
- **校验和**：**强制**，包含伪首部。
- **MSS**：最大报文段长度（不含首部）。
