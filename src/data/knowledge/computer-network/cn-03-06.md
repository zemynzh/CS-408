# 介质访问控制 (CSMA/CD)

## 1. 载波监听多路访问/冲突检测 (CSMA/CD)
- **应用于以太网 (Bus Topology)**。
- **工作步骤**：
  1. **先听后发**：监听总线是否空闲。
  2. **边发边听**：发送时监测是否有碰撞。
  3. **冲突停止**：发现冲突立即停止，并发送干扰信号。
  4. **延迟重发**：等待随机时间（截断二进制指数退避算法）。

## 2. 关键概念
- **争用期 (Contention Period)**：信号在最远两端往返的时间 $2\tau$。
- **最短帧长**：为了确保在发送完之前能检测到冲突。
$$\text{最小帧长} = \text{数据传输率} \times 2\tau$$
- **以太网最小帧长**：64 字节。

## 3. CSMA/CA
- **应用于无线局域网 (WLAN)**。
- **原因**：隐蔽站问题，且无线网卡无法边发边听。
- **机制**：预约信道、ACK 确认、IFS 帧间间隔。
- **RTS/CTS**：请求发送/清除发送，解决隐蔽站问题。
