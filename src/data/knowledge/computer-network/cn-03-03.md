# 流量控制与可靠传输 (停止-等待)

## 1. 流量控制
- **数据链路层**：点对点，限制相邻结点发送速率。
- **传输层**：端到端。

## 2. 停止-等待协议 (Stop-and-Wait)
- **过程**：发送一个帧 -> 等待确认 (ACK) -> 收到 ACK 再发下一个。
- **超时重传**：发送方启动定时器，若超时未收 ACK，则重传。
- **确认丢失**：接收方丢弃重复帧，重传 ACK。
- **信道利用率**：
$$U = \frac{T_D}{T_D + RTT + T_A}$$
- $T_D$：数据帧发送时延。
- $RTT$：往返时延（传播时延 $\times 2$）。
- $T_A$：确认帧发送时延。
- 缺点：信道利用率极低。
