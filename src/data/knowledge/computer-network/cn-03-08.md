# 广域网 (PPP, HDLC)

## 1. 广域网的概念
广域网 (WAN) 覆盖范围广，包含物理层、数据链路层和网络层。

## 2. PPP 协议 (Point-to-Point Protocol)
- **特点**：目前最广泛的点对点链路层协议。面向字节。
- **组成**：
  1. 封装 IP 数据报的方法。
  2. **LCP**：链路控制协议。
  3. **NCP**：网络控制协议。
- **检错**：有 CRC 检错，但不保证可靠传输。

## 3. HDLC 协议 (High-level Data Link Control)
- **特点**：面向位。
- **实现**：采用“0比特填充法”。
- **确认机制**：有序号和确认机制。
