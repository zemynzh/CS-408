# 传输介质

## 1. 导向性传输介质
1. **双绞线 (Twisted Pair)**：
   - 屏蔽双绞线 (STP) 与 非屏蔽双绞线 (UTP)。
   - 常用：5类线、超5类线、6类线。
2. **同轴电缆 (Coaxial Cable)**：
   - 基带同轴电缆 (50Ω)：数字信号。
   - 宽带同轴电缆 (75Ω)：模拟信号（CATV）。
3. **光纤 (Optical Fiber)**：
   - **单模光纤**：直径小，只能传一个模式的光，光源为激光。损耗小，远距离。
   - **多模光纤**：直径大，可传多个模式。光源为 LED。适合近距离。

## 2. 非导向性传输介质
- 无线电波：全向传输。
- 微波：定向传输，如卫星通信（时延大 约270ms）。
- 红外线、蓝牙。

## 3. 物理层接口特性
- **机械特性**：接线器形状、引脚数等。
- **电气特性**：电压范围、阻抗匹配。
- **功能特性**：某线上电压电平的意义。
- **规程特性**：各条线的工作顺序（时序）。
