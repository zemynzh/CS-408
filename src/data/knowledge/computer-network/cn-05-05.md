# TCP 拥塞控制

## 1. 概念
- **流量控制**：点对点。
- **拥塞控制**：全局网络拥塞调控。

## 2. 拥塞窗口 (cwnd)
- 控制发送方在单位时间内能发送的数据量。

## 3. 四种经典算法
1. **慢开始 (Slow Start)**：$cwnd$ 从 1 开始，每经过一个 RTT 指数增长，直到达到阈值 (ssthresh)。
2. **拥塞避免 (Congestion Avoidance)**：$cwnd$ 线性增长（加法增加）。
   - **发生拥塞**（丢包/超时）：$ssthresh = cwnd/2$，$cwnd = 1$，重新慢开始。
3. **快重传 (Fast Retransmit)**：收到 3 个重复确认，认为丢失，立即重传。
4. **快恢复 (Fast Recovery)**：发现丢包（非超时），$ssthresh = cwnd/2$，$cwnd = ssthresh$，进入拥塞避免。
