# 万维网与 HTTP

## 1. 基本概念
- **URL**：统一资源定位符。`<协议>://<主机>:<端口>/<路径>`。
- **HTTP**：超文本传输协议。

## 2. HTTP 协议特点
- 无状态、无连接（虽然底层是 TCP）。
- **版本比较**：
  - **HTTP/1.0**：非持续连接，每个请求需建立 TCP。
  - **HTTP/1.1**：持续连接 (Persistent Connections)。支持流水线 (Pipelining)。
- **方法**：GET, POST, HEAD, PUT, DELETE。

## 3. 响应码
- 200 OK
- 301 Moved Permanently
- 404 Not Found
- 500 Internal Server Error

## 4. Cookie 与 Session
- **Cookie**：存储在浏览器。
- **Session**：存储在服务器，用于追踪会话。
