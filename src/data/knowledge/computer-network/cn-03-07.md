# 以太网与交换机

## 1. 以太网 (Ethernet)
- **拓扑**：逻辑上是总线形，物理上是星形。
- **标准**：IEEE 802.3。
- **帧格式 (MAC 帧)**：
  - 前导码 (8B) + 目的地址 (6B) + 源地址 (6B) + 类型 (2B) + 数据 (46-1500B) + FCS (4B)。
  - **MAC 地址**：48位（6字节），全球唯一。

## 2. 以太网交换机 (Bridge / Switch)
- **原理**：工作在数据链路层。
- **隔离冲突域**：每个端口都是一个独立的冲突域。
- **不隔离广播域**：所有端口属于一个广播域。
- **学习机制**：通过源 MAC 地址构建转发表。
- **交换方式**：
  - **直通交换 (Cut-through)**：只读目的地址即转发，速度快但无差错检测。
  - **存储转发 (Store-and-forward)**：整帧存入缓存并检错，可靠但延迟高。
  - **无碎片转发**：读取前 64 字节确认无碰撞。

## 3. VLAN
- 虚拟局域网。基于 IEEE 802.1Q 协议。
- **作用**：隔离广播域，提高安全性。
