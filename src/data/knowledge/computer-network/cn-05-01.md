# 传输层服务与 UDP

## 1. 传输层基本服务
- **功能**：提供端到端的逻辑通信（进程到进程）。
- **复用与分用**：
  - **复用**：发送端不同进程的数据报使用同一个传输层协议。
  - **分用**：接收端根据端口号交付给不同进程。
- **端口号**：16 位，0~65535。熟知端口 0~1023。

## 2. UDP 协议 (User Datagram Protocol)
- **特点**：无连接、不可靠（尽力而为）、面向报文。
- **优点**：开销小、时延低。支持一对一、一对多、多对多。
- **报文首部**：8 字节（源端口、目的端口、长度、校验和）。
- **校验和**：**可选**，校验首部 + 数据 + 伪首部（含 IP 字段）。
