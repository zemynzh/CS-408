# 物理层设备

## 1. 中继器 (Repeater)
- **原理**：再生数字信号（不是单纯放大）。
- **作用**：扩大网络传输距离。
- **规则**：5-4-3 规则（5个网段、4个中继器、3个网段可挂主机）。
- **局限**：不能隔离冲突域，不能隔离广播域。

## 2. 集线器 (Hub)
- **原理**：多端口的中继器。
- **工作方式**：广播发送，半双工。
- **局限**：**共享带宽**。所有设备处于同一个**冲突域**和**广播域**。
- **性能**：若有 $N$ 个端口，每个端口带宽为 $10Mbps$，则平均可用带宽为 $10/N Mbps$。

## 3. 放大器 (Amplifier)
- **原理**：放大模拟信号。物理层设备主要讨论中继器。
