# 路由协议 (RIP, OSPF, BGP)

## 1. RIP 协议 (Routing Information Protocol)
- **类型**：IGP, 距离向量协议。
- **底层**：基于 UDP (520端口)。
- **距离**：跳数，最大 15，**16 表示不可达**。
- **交换频率**：每 30 秒广播一次所有路由表信息。

## 2. OSPF 协议 (Open Shortest Path First)
- **类型**：IGP, 链路状态协议。
- **底层**：直接封装在 IP 数据报中 (协议号 89)。
- **核心算法**：Dijkstra 算法。
- **交换方式**：洪泛法 (Flooding) 发送链路状态。
- **分层结构**：骨干区域 (Area 0) 与其他区域。

## 3. BGP 协议 (Border Gateway Protocol)
- **类型**：EGP, 路径向量协议。
- **底层**：基于 **TCP (179端口)**，确保传输可靠。
- **目的**：寻找一条可达且无环的路径，而非最佳路径。
