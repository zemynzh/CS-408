# 通信基础

## 1. 基本概念
- **信号**：数据的电气或光学表现。
- **信道**：信号传输的介质。
- **带宽**：信道能通过的最高频率与最低频率之差（Hz），或指单位时间内传输的最大数据量（bps）。

## 2. 奈奎斯特定理 (Nyquist)
在**无噪声**信道中，数据的最大传输速率 $C$：
$$C = 2W \log_2 V \text{ (bps)}$$
- $W$：信道带宽 (Hz)。
- $V$：码元的离散电平数目。
- 结论：为了提高数据率，必须设法使每个码元携带更多比特的信息。

## 3. 香农定理 (Shannon)
在**有噪声**信道中，信道的极限信息传输速率 $C$：
$$C = W \log_2(1 + S/N) \text{ (bps)}$$
- $S$：信号功率。
- $N$：噪声功率。
- **信噪比 (dB)**：$10 \log_{10}(S/N)$。
- 结论：只要信息传输速率低于极限速率，就能实现无差错传输。

## 4. 编码与调制
### 数字数据编码为数字信号
- **不归零编码 (NRZ)**：容易丢同步。
- **曼彻斯特编码 (Manchester)**：位中间有跳转。既作为时钟信号，又作为数据信号。1：高到低，0：低到高（或相反）。**效率 50%**。
- **差分曼彻斯特编码**：位开始有跳转表示0，无跳转表示1。抗干扰强。
### 数字数据调制为模拟信号
- ASK（幅移键控）、FSK（频移键控）、PSK（相移键控）、QAM（正交幅度调制）。

## 5. 电路交换、报文交换与分组交换
- **分组交换**：采用存储转发技术，时延小，不独占链路。
- **数据报**：无连接，尽力而为。
- **虚电路**：面向连接，逻辑上的专用连接。
