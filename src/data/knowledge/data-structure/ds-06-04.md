# ds-06-04: 图的应用（最小生成树、最短路径）

## 1. 最小生成树 (MST)
对于带权连通无向图，WPL 最小的生成树。

### (1) Prim 算法
- **逻辑**：从某个顶点开始，每次将距离当前**生成树集合**最近的顶点加入集合，直到所有顶点加入。
- **复杂度**：$O(n^2)$，适合**稠密图**。

### (2) Kruskal 算法
- **逻辑**：按边权从小到大排序，每次选择不形成回路的边加入，直到选够 $n-1$ 条边。
- **复杂度**：$O(e \log e)$，适合**稀疏图**。

## 2. 最短路径
### (1) Dijkstra 算法 (单源最短路)
- **逻辑**：
    1. 初始化 $dist$ 数组为 $\infty$，起始点为 0。
    2. 选择 $dist$ 最小的未确定顶点 $u$。
    3. 以 $u$ 为中转点，松弛（Relax）其邻接点 $v$：若 $dist[u] + w(u,v) < dist[v]$，则更新 $dist[v]$。
    4. 重复直至所有点确定。
- **限制**：**不能处理带负权值的边**。
- **复杂度**：$O(n^2)$。

### (2) Floyd 算法 (各对顶点间最短路)
- **逻辑**：动态规划。$A^k[i][j] = \min(A^{k-1}[i][j], A^{k-1}[i][k] + A^{k-1}[k][j])$。
- **复杂度**：$O(n^3)$。
- **优点**：可以处理带负权边的图，但不能处理带负权回路的图。
