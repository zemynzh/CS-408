# 2.1 线性表的定义和基本操作

### 1. 线性表的定义
线性表是具有相同数据类型的 $n$ ($n \ge 0$) 个数据元素的有限序列。
- **$n$**：表长。当 $n=0$ 时，线性表为空表。
- **$a_i$**：线性表中的第 $i$ 个元素，称为线性表的**位序**（注意：位序从 1 开始，数组下标从 0 开始）。
- **特点**：
  - 元素个数有限。
  - 逻辑上相邻，物理上不一定相邻。
  - 数据类型相同，占用相同大小存储空间。

### 2. 线性表的基本操作
- `InitList(&L)`：初始化表。
- `Length(L)`：求表长。
- `LocateElem(L, e)`：按值查找。
- `GetElem(L, i)`：按位查找。
- `ListInsert(&L, i, e)`：插入操作。
- `ListDelete(&L, i, &e)`：删除操作。
- `PrintList(L)`：输出操作。
- `Empty(L)`：判空操作。
- `DestroyList(&L)`：销毁操作。

> **考点提示**：
> 1. 线性表是一种**逻辑结构**，顺序表和链表是其**存储结构**。
> 2. 基本操作的实现取决于采用何种存储结构。
