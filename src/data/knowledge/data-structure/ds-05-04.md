# ds-05-04: 二叉树的遍历和线索二叉树

## 1. 二叉树的遍历
遍历是指按某条搜索路径巡访树中每个结点，且每个结点只被访问一次。

### (1) 先序遍历 (PreOrder)
- 访问根结点 $\rightarrow$ 遍历左子树 $\rightarrow$ 遍历右子树。
- 逻辑：`Visit(root); PreOrder(root->lchild); PreOrder(root->rchild);`

### (2) 中序遍历 (InOrder)
- 遍历左子树 $\rightarrow$ 访问根结点 $\rightarrow$ 遍历右子树。
- 逻辑：`InOrder(root->lchild); Visit(root); InOrder(root->rchild);`
- **重要特性**：二叉排序树的中序遍历序列是有序的。

### (3) 后序遍历 (PostOrder)
- 遍历左子树 $\rightarrow$ 遍历右子树 $\rightarrow$ 访问根结点。
- 逻辑：`PostOrder(root->lchild); PostOrder(root->rchild); Visit(root);`

### (4) 层序遍历
- 使用**队列**实现。从根开始，访问一个结点后，将其左右孩子（若有）入队。

## 2. 线索二叉树 (Threaded Binary Tree)
### (1) 背景与定义
二叉链表中有 $n+1$ 个空指针。利用这些空指针存放结点在某种遍历序列下的前驱和后继信息。
- 若 `lchild` 为空，令其指向前驱；若 `rchild` 为空，令其指向后继。
- 增加两个标志位 `ltag` 和 `rtag`：
    - `tag == 0`：指针指向孩子。
    - `tag == 1`：指针指向线索（前驱/后继）。

### (2) 中序线索化算法逻辑
```cpp
void InThread(ThreadTree &p, ThreadTree &pre) {
    if (p != NULL) {
        InThread(p->lchild, pre); // 递归线索化左子树
        if (p->lchild == NULL) {  // 建立前驱线索
            p->lchild = pre;
            p->ltag = 1;
        }
        if (pre != NULL && pre->rchild == NULL) { // 建立后继线索
            pre->rchild = p;
            pre->rtag = 1;
        }
        pre = p;
        InThread(p->rchild, pre); // 递归线索化右子树
    }
}
```

### (3) 优缺点
- 优点：找前驱、后继快；遍历不需要栈。
- 缺点：插入、删除操作逻辑变复杂。
