# ds-05-02: 二叉树的定义及其主要特性

## 1. 二叉树的定义
二叉树是 $n (n \ge 0)$ 个结点的有限集合：
1. 或者为空二叉树。
2. 或者由一个根结点和两个互不相交的被称为左子树和右子树的二叉树组成。

**二叉树与度为 2 的有序树的区别**：
- 二叉树可以为空，而度为 2 的树至少有 3 个结点。
- 二叉树的子树有左右之分，即使只有一棵子树也要区分左、右。

## 2. 特殊二叉树
- **满二叉树**：高度为 $h$，含有 $2^h - 1$ 个结点的二叉树。每层结点都达到最大值。
- **完全二叉树**：高度为 $h$，含有 $n$ 个结点，每个结点都与高度为 $h$ 的满二叉树中编号为 $1 \sim n$ 的结点一一对应。
    - *特点*：只有最后两层可能有叶子，叶子都在左侧连续出现。
- **二叉排序树（BST）**：左子树关键字 < 根结点 < 右子树关键字。
- **平衡二叉树（AVL）**：任一结点的左、右子树深度之差不超过 1。

## 3. 二叉树的性质
1. **非空二叉树上的叶子结点数等于度为 2 的结点数加 1**，即 $n_0 = n_2 + 1$。
    - *证明*：总结点数 $n = n_0 + n_1 + n_2$；总边数 $B = n - 1$；且 $B = n_1 + 2n_2$。联立得 $n_0 + n_1 + n_2 - 1 = n_1 + 2n_2 \Rightarrow n_0 = n_2 + 1$。
2. **非空二叉树第 $i$ 层上至多有 $2^{i-1}$ 个结点 ($i \ge 1$)**。
3. **高度为 $h$ 的二叉树至多有 $2^h - 1$ 个结点 ($h \ge 1$)**。
4. **具有 $n$ 个结点的完全二叉树的高度为 $\lceil \log_2(n+1) \rceil$ 或 $\lfloor \log_2 n \rfloor + 1$**。
5. **对完全二叉树按层序编号（根为 1）**：
    - 若 $i > 1$，双亲结点编号为 $\lfloor i/2 \rfloor$。
    - 若 $2i \le n$，左孩子编号为 $2i$，否则无左孩子。
    - 若 $2i+1 \le n$，右孩子编号为 $2i+1$，否则无右孩子。
