# 3.2 队列 (Queue)

### 1. 定义
只允许在表的一端进行插入（入队），在另一端进行删除（出队）的线性表。
- **队头 (Front)**：删除端。
- **队尾 (Rear)**：插入端。
- **特性**：先进先出 (**FIFO**, First In First Out)。

### 2. 队列的顺序存储 (循环队列)
为了解决“假溢出”问题，将顺序表在逻辑上视为一个环。
- **判空**：`Q.front == Q.rear`
- **判满**（牺牲一个单元）：`(Q.rear + 1) % MaxSize == Q.front`
- **队列长度**：`(Q.rear - Q.front + MaxSize) % MaxSize`

### 3. 队列的链式存储 (链队列)
同时包含头指针 `front` 和尾指针 `rear` 的单链表。
- 出队：删除头结点的后继。
- 入队：在尾结点后插入并更新 `rear`。

### 4. 双端队列 (Deque)
允许两端都可以进行入队和出队。
- **输出受限的双端队列**：一端入/出，另一端仅入。
- **输入受限的双端队列**：一端入/出，另一端仅出。

> **常考点**：根据进队序列判断不合法的出队序列。
