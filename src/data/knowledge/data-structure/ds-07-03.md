# 7.3 树形查找

### 1. 二叉排序树 (BST)
- **定义**：左子树 < 根 < 右子树。
- **操作**：
  - **查找/插入**：类似折半查找。
  - **删除**：若为叶子直接删；若只有一棵子树，子承父业；若有两棵子树，用中序前驱或中序后继替代。
- **效率**：取决于树的高度。最好 $O(\log_2 n)$，最坏 $O(n)$（单支树）。

### 2. 平衡二叉树 (AVL)
- **定义**：BST 基础上，每个结点的左、右子树高度差绝对值不超过 1。
- **平衡因子 (BF)**：左子树高度 - 右子树高度。
- **平衡旋转**：LL、RR、LR、RL 旋转。

### 3. 红黑树 (Red-Black Tree)
为了降低 AVL 频繁旋转的开销，一种弱平衡二叉树。
- **五大性质**：
  1. 结点非黑即红。
  2. 根结点是黑色。
  3. 叶结点（NIL）是黑色。
  4. 红结点的孩子必为黑色。
  5. 任意结点到叶结点的简单路径包含相同黑结点。
- **结论**：红黑树高度 $h \le 2\log_2(n+1)$。

> **考点**：BST 的中序遍历序列是有序的。
