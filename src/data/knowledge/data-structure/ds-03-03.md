# 3.3 栈和队列的应用

### 1. 栈的应用
- **括号匹配**：利用栈检测左右括号是否匹配。
- **表达式求值**：
  - **中缀转后缀 (逆波兰式)**：利用栈暂存运算符。
  - **后缀表达式求值**：遇到数字压栈，遇到运算符弹出两个操作数计算后再压栈。
- **递归**：递归调用时，系统通过栈来保存函数的局部变量和返回地址。

### 2. 队列的应用
- **层次遍历**：在树或图的 BFS（广度优先搜索）中利用队列保存待访问节点。
- **计算机系统应用**：
  - 打印机数据缓冲区（先来先服务）。
  - 多队列调度策略。

> **补充**：函数调用栈（Stack Frame）存储了返回地址、实参和局部变量。
