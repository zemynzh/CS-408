# 7.4 B 树和 B+ 树

### 1. B 树 (多路平衡查找树)
适用于磁盘等外部存储。
- **$m$ 阶 B 树的性质**：
  - 每个结点最多 $m$ 棵子树，$m-1$ 个关键字。
  - 根结点若非叶子，至少 2 棵子树。
  - 非根结点至少 $\lceil m/2 \rceil$ 棵子树。
  - 所有叶结点都在同一层。

### 2. B+ 树
B 树的一种变形，广泛用于数据库索引。
- **区别于 B 树**：
  - 非叶结点只存关键字和指针，不存数据。
  - 所有数据都存储在叶结点中。
  - 叶结点之间有指针相连（支持顺序查找）。
  - 分支结点关键字 $n$ 对应 $n$ 棵子树（B 树是 $n+1$）。

> **考点**：B 树的插入（分裂）和删除（借用、合并）过程。
