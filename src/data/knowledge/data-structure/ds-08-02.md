# 8.2 插入排序

### 1. 直接插入排序 (Straight Insertion Sort)
将待排序记录插入到前面已排好的子序列中。
- **时间**：最好 $O(n)$，最坏 $O(n^2)$。
- **空间**：$O(1)$。
- **稳定性**：稳定。

### 2. 折半插入排序
利用折半查找确定插入位置，减少比较次数。
- **时间**：$O(n^2)$。比较次数减少，但移动次数未变。
- **稳定性**：稳定。

### 3. 希尔排序 (Shell Sort)
缩小增量排序。将记录按步长 $d$ 分组。
- **时间**：通常 $O(n^{1.3})$，最坏 $O(n^2)$。
- **空间**：$O(1)$。
- **稳定性**：**不稳定**。
