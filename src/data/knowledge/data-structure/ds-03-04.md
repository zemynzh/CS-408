# 3.4 数组和特殊矩阵

### 1. 数组的存储
数组通常采用**顺序存储**。
- **行优先存储**：先存第一行，再存第二行。
- **列优先存储**：先存第一列，再存第二列。

### 2. 特殊矩阵的压缩存储
#### (1) 对称矩阵 ($a_{ij} = a_{ji}$)
只存下三角（含对角线）部分，存入一维数组 $B$ 中。
- 元素 $a_{ij}$ ($i \ge j$) 在 $B$ 中的下标 $k = \frac{i(i-1)}{2} + j - 1$ (下标从 0 开始)。

#### (2) 三角矩阵
上三角或下三角为常数。只需额外增加一个位置存储常数。

#### (3) 三对角矩阵 (带状矩阵)
只有 $|i-j| \le 1$ 时有非零值。
- 元素 $a_{ij}$ 在一维数组中的位置 $k = 2i + j - 3$ (下标从 1 开始)。

### 3. 稀疏矩阵
非零元素个数远小于总元素个数。
- **存储方式**：
  - **三元组表**：存储 `(行, 列, 值)`。失去随机存取特性。
  - **十字链表**：适用于频繁增删。

> **考点**：计算压缩存储后元素在一维数组中的偏移量。
