# 7.5 散列 (Hash) 查找

### 1. 基本概念
- **散列函数**：将关键字映射到地址的函数 $H(key)$。
- **冲突**：不同关键字映射到同一地址。
- **同义词**：发生冲突的两个关键字。

### 2. 散列函数构造
- **除留余数法**：$H(key) = key \% p$。$p$ 通常取不大于表长且最接近的质数。

### 3. 冲突处理方法
- **开放定址法**：
  - **线性探测法**：冲突后看下一个，容易产生“聚集”。
  - **平方探测法**：$+1^2, -1^2, +2^2...$
- **拉链法 (Chaining)**：冲突的元素存入对应的单链表中。

### 4. 散列查找效率
取决于：散列函数、冲突处理方法、**装填因子 ($\alpha$)**。
- $\alpha = \text{表中记录数} / \text{散列表长度}$。
- $\alpha$ 越大，冲突概率越高。
