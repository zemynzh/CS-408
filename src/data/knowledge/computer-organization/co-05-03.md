# 数据通路 (co-05-03)

## 1. 基本概念
- **数据通路**：指令执行过程中，数据在 CPU 内部流动的路径。
- **主要逻辑**：寄存器传送、寄存器与 ALU 传送、存储器读写。

## 2. 单总线结构
- 简单但冲突多，同一时刻只能有一组数据在线上传送。
- CPU 内部只有一个公共总线。

## 3. 多总线结构
- 双总线、三总线：ALU 输出和输入分别接不同总线，减少冲突，加快运算速度。

## 4. 重点控制信号
- $R_{in}$ / $R_{out}$：寄存器读入/读出信号。
- $M_{in}$ / $M_{out}$：MDR 数据传送。
- $Read$ / $Write$：读/写主存。
- $Add, Sub, \dots$：ALU 操作。
- $PC_{out}, MAR_{in}$：取指起始。
