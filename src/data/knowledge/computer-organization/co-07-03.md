# 程序中断方式 (co-07-03)

## 1. 中断的基本概念
- **中断请求 (IRQ)**：外设请求 CPU 服务。
- **中断响应条件**：
  - CPU 处于开中断状态 (IF=1)。
  - 当前指令执行结束。
  - 中断源有请求且未被屏蔽。

## 2. 中断处理过程 (重点)
1. **中断响应**：
  - 关中断。
  - 保护断点 (PC 压栈)。
  - 寻找中断服务程序入口 (向量地址)。
2. **执行中断服务程序 (ISR)**：
  - 保护现场 (GPRs 压栈)。
  - 执行具体 I/O 服务。
  - 恢复现场。
  - 开中断。
  - 中断返回 (IRET)。

## 3. 核心机制
- **向量地址**：存储在向量表中的 ISR 首地址。
- **多重中断 (Nested Interrupt)**：
  - 优先级高的中断可以打断优先级低的中断。
  - **中断屏蔽码**：通过屏蔽码动态改变中断优先级。
