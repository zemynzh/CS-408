# CPU 的功能与结构 (co-05-01)

## 1. CPU 的基本功能
- **指令控制**：程序执行的先后顺序。
- **操作控制**：向各部件发出控制信号。
- **时间控制**：各种操作实施时间的定时。
- **数据加工**：对数据进行算术逻辑运算。

## 2. 核心寄存器
- **PC (Program Counter)**：存放下一条指令地址。
- **IR (Instruction Register)**：存放当前执行的指令。
- **MAR/MDR**：存储器地址和数据缓冲。
- **PSW (Program Status Word)**：程序状态字，标志位 (CF, ZF, SF, OF 等)。
- **SP (Stack Pointer)**：栈指针。
- **GPRs (General Purpose Registers)**：通用寄存器。

## 3. 运算器 (ALU)
- 完成逻辑运算 ($AND, OR, NOT$) 和算术运算 ($+, -, \times, \div$)。

## 4. 控制器 (CU)
- 负责译码及产生微操作序列。
- 包含：微命令、微操作、微指令。
- **硬布线** vs **微程序** 控制。
| 类别 | 硬布线 (Combinational) | 微程序 (Micro-programmed) |
| :--- | :--- | :--- |
| 控制信号 | 逻辑门/状态机 | 微指令 (存储在 ROM) |
| 速度 | 快 | 慢 (访存) |
| 复杂性 | 复杂 | 结构规整 |
| 扩充性 | 差 (硬修改) | 好 (软件修改) |
