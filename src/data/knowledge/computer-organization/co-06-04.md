# 总线操作与定时 (co-06-04)

## 1. 总线定时 (Timing)
确保主从设备在时序上保持一致。

## 2. 同步定时 (Synchronous)
- **特点**：由统一的时钟信号控制。
- **优点**：简单，逻辑少。
- **缺点**：速度由最慢设备决定，且受总线长度限制。

## 3. 异步定时 (Asynchronous)
- **特点**：通过“应答”信号 (握手) 进行数据交换。
- **握手方式**：
  1. **不互锁 (Non-Interlocked)**：主设备发请求，不等待响应。
  2. **半互锁 (Semi-Interlocked)**：主设备发请求，等待响应，收到后再撤销。
  3. **全互锁 (Fully-Interlocked)**：主设备收到响应后再撤销，从设备收到主撤销后再撤销。最可靠。

## 4. 重点结论
- 异步定时适合处理速度差异较大的设备。
- **突发传输 (Burst Transfer)**：一次握手传送多个连续地址的数据。
