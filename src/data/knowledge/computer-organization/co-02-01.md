# 进位计数制与码制 (co-02-01)

## 1. 进位计数制
- **二进制 (B)**: $\{0, 1\}$
- **八进制 (O)**: $\{0, 1, 2, 3, 4, 5, 6, 7\}$
- **十进制 (D)**: $\{0, 1, \dots, 9\}$
- **十六进制 (H)**: $\{0, 1, \dots, 9, A, B, C, D, E, F\}$

## 2. 转换方法
- **十进制 $\to$ $r$ 进制**：
  - 整数部分：除 $r$ 取余法 (由低位向高位排列)。
  - 小数部分：乘 $r$ 取整法 (由高位向低位排列)。
- **$r$ 进制 $\to$ 十进制**：
  - 按权展开：$\sum a_i \times r^i$。

## 3. 码制 (Codes)
- **BCD 码 (Binary-Coded Decimal)**:
  - 8421 码：权值为 8, 4, 2, 1。
  - 余 3 码：8421 码 $+ (0011)_2$。
  - 2421 码：权值为 2, 4, 2, 1 (大数从 $1010$ 开始)。
- **ASCII 码**：7 位二进制表示 128 个字符。
- **校验码**：
  - 奇偶校验：只能检测奇数个错误。
  - 海明校验 (Hamming Code)：能检测两位错，纠正一位错。计算公式：$2^k \ge n + k + 1$。
  - 循环冗余校验 (CRC)：模 2 除法，多项式除法。
