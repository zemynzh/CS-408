# 总线仲裁 (co-06-03)

## 1. 仲裁的基本概念
- **主设备 (Master)**：具有主动申请并控制总线能力的设备。
- **从设备 (Slave)**：响应主设备请求的设备。
- **仲裁 (Arbitration)**：多个主设备竞争时，决定总线所有权的机制。

## 2. 集中仲裁 (Centralized)
由总线控制器统一处理。
1. **链式查询 (Daisy Chain)**：结构简单，故障敏感。按物理连接顺序优先级排序 (近的高)。
2. **计数器定时查询 (Polling)**：控制器通过计数器轮流轮询。可变优先级。
3. **独立请求 (Independent Request)**：每个设备有独立的请求/授权线。速度快，线路多。

## 3. 分布式仲裁 (Distributed)
没有统一控制器，每个设备自带仲裁逻辑。
- 逻辑竞争方式。
- 令牌传递。
- **自举逻辑**。
