# 定点运算 (co-02-03)

## 1. 移位运算
- **算术移位**：
  - 正数：补 0。
  - 负数 (原码)：补 0。
  - 负数 (反码)：补 1。
  - 负数 (补码)：左移补 0，右移补 1。
- **逻辑移位**：不分正负，统一补 0。
- **循环移位**：包含进位位移位和大循环移位。

## 2. 加减法运算
- **补码加减法公式**：
  - $[X+Y]_{\text{补}} = [X]_{\text{补}} + [Y]_{\text{补}}$
  - $[X-Y]_{\text{补}} = [X]_{\text{补}} + [-Y]_{\text{补}}$
- **溢出判断**：
  - **双符号位法 (变形补码)**：结果符号位 $01$ 表示正溢出，$10$ 表示负溢出。
  - **进位判断法**：符号位进位 $C_s$ 与数值位最高位进位 $C_{n-1}$ 异或：$V = C_s \oplus C_{n-1} = 1$ 则溢出。

## 3. 乘除法运算
- **原码一位乘 (比较常用)**：类似手工乘法，符号位异或处理。
- **补码一位乘 (Booth 算法)**：
  - 根据 $[Y_{i}] - [Y_{i+1}]$ 判断操作 (00, 11 不变；01 加 $[X]_{\text{补}}$；10 加 $[-X]_{\text{补}}$)。
  - 右移为算术右移。
- **除法**：原码恢复余数法、加减交替法 (不恢复余数法)。
