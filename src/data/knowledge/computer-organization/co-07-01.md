# I/O 系统概述 (co-07-01)

## 1. I/O 系统结构
- **I/O 接口**：连接主机与外设，负责信号转换和数据缓冲。
- **I/O 总线**：数据、地址、控制信号。
- **I/O 端口**：接口内部可由 CPU 寻址的寄存器。
  - 数据端口 (Read/Write Data)
  - 状态端口 (Busy, Ready)
  - 控制端口 (Start, Stop)

## 2. 编址方式
- **统一编址 (Memory-Mapped I/O)**：I/O 端口与内存统一地址空间。指令丰富，但占用内存地址。
- **独立编址 (Isolated I/O)**：使用专门的 I/O 指令 (如 `IN`, `OUT`)。

## 3. 控制方式演进
- **程序查询方式** (Polling)
- **程序中断方式** (Interrupt)
- **DMA 方式** (Direct Memory Access)
- **通道方式** (Channel)
