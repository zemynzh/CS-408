# 总线性能指标 (co-06-02)

## 1. 核心指标
- **总线宽度**：数据线的根数 (如 32 位)。
- **总线时钟周期**：完成一次基本动作的时间。
- **总线时钟频率**：1/周期。
- **总线工作频率**：单位时间内传输次数。
- **总线带宽 (Bus Bandwidth)**：单位时间内总线传输的最大字节数。
  $$ \text{带宽} = \frac{\text{总线宽度}}{8} \times \text{工作频率} \text{ (B/s)} $$

## 2. 总线复用 (Multiplexing)
- 同一组线在不同时间段传输不同的信息 (如地址和数据复用)。
- 目的：减少引脚数和布线空间。

## 3. 常见总线标准
- **ISA / EISA / PCI / PCIe**
- **USB / SATA / SCSI**
