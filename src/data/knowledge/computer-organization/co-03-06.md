# 虚拟存储器 (co-03-06)

## 1. 核心概念
- 将主存与辅存逻辑上统一。用户程序空间大于主存物理空间。
- **辅存地址**：由磁盘地址 (柱面、磁头、扇区) 确定。

## 2. 映射方式
- **页式虚拟存储 (Paging)**：主存和程序划分为固定大小的页。
  - **页表 (Page Table)**：存储逻辑页号到物理页框号的对应。
  - **快表 (TLB)**：存储最近使用的页表项 (Cache of Page Table)。
- **段式虚拟存储 (Segmentation)**：按逻辑段划分，长度不固定。
- **段页式虚拟存储**：先分段再分页。

## 3. 地址翻译过程
1. **逻辑地址** $\to$ **TLB 查找**。
2. TLB 未命中 $\to$ **Page Table 查找** (主存)。
3. Page Table 未命中 $\to$ **缺页中断 (Page Fault)** (调入辅存数据)。
4. 物理页号 + 页内偏移 $\to$ **物理地址**。

## 4. 常见问题
- 虚拟存储器由**硬件和操作系统**共同管理。
- 缺页次数与主存块数的关系 (Belady 异常)。
