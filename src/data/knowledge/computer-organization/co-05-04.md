# 控制器设计 (co-05-04)

## 1. 硬布线控制器 (Hardwired Control)
- 利用布线逻辑实现逻辑电路。
- **特点**：速度快，用于 RISC。
- **设计**：通过状态图、节拍划分。

## 2. 微程序控制器 (Micro-programmed Control)
- 将控制信号编制成**微指令**，存储在**控制存储器 (CS)** 中。
- **核心组件**：微指令寄存器 ($uIR$)、控制存储器 ($CM$)、微地址形成逻辑。
- **微指令格式**：
  - **水平型**：一位对一个控制信号，并行性好，微指令字长大。
  - **垂直型**：类似机器指令格式，编译后执行，微指令短，速度慢。
  - **混合型**：折中。

## 3. 编码方式
- **直接编码 (不译码)**：直接输出位。
- **字段译码**：分段译码，减少位数 (相斥信号放一起)。
- **字段直接编码**。

## 4. 下地址形成
- 由下地址字段给出。
- 根据标志位、操作码映射 (Opcode $\to$ Address)。
