# Cache 存储器 (co-03-05)

## 1. Cache 基本原理
- **局部性原理**：时间局部性、空间局部性。
- **命中率 ($h$)**：$h = N_c / (N_c + N_m)$。
- **平均存取时间**：$T_a = h \times t_c + (1-h) \times t_m$。

## 2. 地址映射方式 (重点)
1. **直接映射 (Direct Mapping)**：主存块号 $j$ 映射到 Cache 块号 $i = j \mod C$ ($C$ 为 Cache 块数)。
2. **全相联映射 (Fully Associative)**：主存块可放置在 Cache 任意位置。冲突少，查找慢。
3. **组相联映射 (Set Associative)**：$i = j \mod Q$ ($Q$ 为 Cache 组数)。兼具两者优点。
  - 主存地址格式：`主存字块标记 | 组号 | 块内地址`。

## 3. 替换算法
- **FIFO** (先进先出)
- **LRU** (最近最少使用) — 命中率最高。
- **LFU** (最不经常使用)
- **RAND** (随机)

## 4. 写策略
- **写回法 (Write Back)**：仅当 Cache 块被替换时才写回主存 (需脏位 Dirty Bit)。
- **全写法 (Write Through)**：同时写 Cache 和主存。
