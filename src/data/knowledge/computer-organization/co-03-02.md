# 主存储器 (co-03-02)

## 1. SRAM vs DRAM
- **SRAM (Static RAM)**：使用双稳态触发器存储，速度快，功耗高，集成度低，用于 Cache。
- **DRAM (Dynamic RAM)**：使用电容存储电荷，速度慢，功耗低，集成度高，用于主存。需刷新。

## 2. DRAM 刷新方式
- **集中刷新**：刷新时所有存储单元暂停读写 (死时间)。
- **分散刷新**：将工作周期分为读写和刷新两部分 (无死时间，但增加存取时间)。
- **异步刷新**：在最大刷新时间内，按行分散安排刷新 (死时间小，性能均衡)。

## 3. 存储器地址扩展
- **位扩展**：并行输出更多位 (地址线不变，数据线增加)。
- **字扩展**：增加地址深度 (地址线增加，数据线不变)。使用片选信号 (Decoder)。

## 4. 读写周期
- **读周期**：地址稳定 $\to$ 片选 $\to$ 数据就绪 $\to$ 数据失效。
- **写周期**：地址稳定 $\to$ 数据稳定 $\to$ 写信号使能 $\to$ 完成写操作。
