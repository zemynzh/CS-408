# 存储器概述 (co-03-01)

## 1. 存储器的分类
- **按作用分**：
  - 高速缓冲存储器 (Cache)
  - 主存储器 (Main Memory)
  - 辅助存储器 (Auxiliary Memory)
- **按介质分**：
  - 半导体存储器 (RAM, ROM)
  - 磁性存储器 (Disk, Tape)
  - 光存储器 (CD, DVD)
- **按存取方式分**：
  - 随机存取存储器 (RAM): SRAM, DRAM
  - 顺序存取存储器 (SAM): 磁带
  - 直接存取存储器 (DAM): 硬盘

## 2. 存储器的多级层次结构
- 缓存-主存 (主要解决速度不匹配)
- 主存-辅存 (主要解决容量不足)

## 3. 性能指标
- **存储容量**：$M \times N$ (存储单元数 $\times$ 存储字长)。
- **存取周期 ($T_m$)**：两次独立的存储器操作所需的最短时间间隔。
- **存储带宽 ($B$)**：单位时间内传送的二进制位数。
  $$ B = \frac{\text{总线宽度}}{T_m} $$
