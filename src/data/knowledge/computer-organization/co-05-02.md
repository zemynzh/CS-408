# 指令执行过程 (co-05-02)

## 1. 指令周期 (Instruction Cycle)
- **定义**：CPU 取出并执行一条指令所需的时间。
- **阶段划分**：
  1. **取指阶段 (Fetch)**：$PC \to MAR \to Memory \to MDR \to IR$。
  2. **间址阶段 (Indirect)**：若指令中有间址位，需访存获取有效地址。
  3. **执行阶段 (Execute)**：ALU 运算。
  4. **中断阶段 (Interrupt)**：检查并响应中断。

## 2. 机器周期与时钟周期
- **指令周期** $\ge$ **机器周期** $\ge$ **时钟周期**。
- 一个机器周期通常包含若干个时钟周期。

## 3. 指令流水线
- **并行性**：在一个时钟周期内，不同指令的不同阶段重叠执行。
- **目标**：提高吞吐率 (Throughput)。
