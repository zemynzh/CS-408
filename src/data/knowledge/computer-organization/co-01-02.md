# 计算机系统层次结构 (co-01-02)

## 1. 冯·诺依曼架构 (Von Neumann Architecture)
### 核心思想
- **存储程序**：程序和数据以二进制形式存储在存储器中，由 CPU 自动执行。
- **五大部件**：运算器、控制器、存储器、输入设备、输出设备。
- **以运算器为中心** (现代架构多以存储器为中心)。
- **指令和数据同等地位**：存储在同一存储器中，按地址寻址。

## 2. 计算机硬件系统
- **中央处理器 (CPU)** = 运算器 + 控制器
- **存储器**：主存 (RAM) 和辅存。
- **I/O 设备**：输入和输出。

## 3. 计算机系统的层次结构
计算机系统通常被划分为 5 个层次（从低到高）：
1. **微程序级 (M0)**：由微指令执行，硬件级。
2. **机器语言级 (M1)**：传统机器语言，硬件/固件级。
3. **操作系统级 (M2)**：由操作系统指令组成。
4. **汇编语言级 (M3)**：由汇编语言组成。
5. **高级语言级 (M4)**：由高级语言（C/C++、Java 等）编写。

## 4. 关键概念
- **机器字长**：CPU 内部寄存器的位数，通常也是运算器进行定点运算的位数。
- **指令字长**：一条指令包含的二进制代码的位数。
- **存储字长**：一个存储单元存储的二进制代码的长度。
- **MAR/MDR**：
    - **MAR (Memory Address Register)**：存储单元地址寄存器。
    - **MDR (Memory Data Register)**：存储单元数据寄存器。
