# 寻址方式 (co-04-02)

## 1. 指令寻址
- **顺序寻址**：$(PC) + 1 \to PC$。
- **跳跃寻址**：通过 JMP、CALL 指令修改 PC 的值。

## 2. 数据寻址 (重点)
设指令中地址码字段为 $A$，有效地址为 $EA$，$R$ 为寄存器内容：
1. **立即寻址**：$A$ 即为操作数。最快。
2. **直接寻址**：$EA = A$。访存 1 次。
3. **间接寻址**：$EA = (A)$。访存 2 次以上。
4. **寄存器寻址**：$EA = R_i$。不访存，速度快。
5. **寄存器间接寻址**：$EA = (R_i)$。访存 1 次。
6. **偏移寻址**：
   - **相对寻址**：$EA = (PC) + A$。有利于程序浮动。
   - **基址寻址**：$EA = (BR) + A$。有利于程序分段和扩充编址空间。
   - **变址寻址**：$EA = (IX) + A$。有利于处理数组和循环。
7. **堆栈寻址**：$EA = SP$。

## 3. 计算总结
- $A$：形式地址。
- $(X)$：存储在地址 $X$ 处的内容。
