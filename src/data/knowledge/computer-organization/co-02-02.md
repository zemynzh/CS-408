# 定点数的表示 (co-02-02)

## 1. 有符号数编码
设机器字长为 $n$ 位 (1 位符号位，$n-1$ 位数值位)：
- **原码 $[X]_{\text{原}}$**：符号位 (0 正 1 负) $+$ 数值绝对值。
- **反码 $[X]_{\text{反}}$**：
  - 正数：同原码。
  - 负数：符号位不变，数值位按位取反。
- **补码 $[X]_{\text{补}}$**：
  - 正数：同原码。
  - 负数：反码 $+ 1$ (即从原码最后一位 1 开始，左边的全部取反)。
  - **模 2 补码**：用于算术运算。
- **移码 $[X]_{\text{移}}$**：仅用于浮点数阶码。$[X]_{\text{移}} = X + 2^n$ (即补码符号位取反)。

## 2. 表示范围 (整数)
- **$n$ 位原码/反码**：$[-(2^{n-1}-1), 2^{n-1}-1]$
- **$n$ 位补码**：$[-2^{n-1}, 2^{n-1}-1]$
  - 补码中，$-(2^{n-1})$ 的表示为 $10\dots0$。

## 3. 核心性质
- **补码的补码**：$[[X]_{\text{补}}]_{\text{补}} = [X]_{\text{原}}$ (误，应为 $X$)。实际上 $[[X]_{\text{补}}]_{\text{补}} = [X]_{\text{补}}$ (不对，应为：若 $X$ 的补码再求一次补码，则得到 $X$ 本身对应的二进制位序列)。更正：补码求补 (变补) 得到其相反数的补码。
- **零的表示**：
  - 原码/反码：$\pm 0$。
  - 补码/移码：**唯一** $0$。
