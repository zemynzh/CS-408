# 浮点数表示与运算 (co-02-04)

## 1. 浮点数格式
$$ N = M \times R^E $$
- $M$：尾数 (Mantissa)，通常为补码或原码小数。
- $E$：阶码 (Exponent)，通常为移码或补码整数，决定范围。
- $R$：基数，通常为 2。

## 2. IEEE 754 标准
- **单精度 (32 位)**：1 位符号位 (S) + 8 位阶码 (E) + 23 位尾数 (M)。
  - 偏移量 (Bias) = 127。
  - 尾数隐藏最高位 1，实际有效位 24 位。
- **双精度 (64 位)**：1 位符号位 (S) + 11 位阶码 (E) + 52 位尾数 (M)。
  - 偏移量 (Bias) = 1023。

## 3. 规格化 (Normalization)
- **原码规格化**：尾数最高位为 1。格式：$0.1\dots$ 或 $1.1\dots$。
- **补码规格化**：符号位与最高数值位不同。格式：$0.1\dots$ 或 $1.0\dots$。
  - 特例：$-1/2$ 的补码规格化形式为 $1.0\dots0$。

## 4. 浮点加减运算步骤
1. **对阶**：小阶向大阶看齐。尾数右移，阶码增加。
2. **尾数求和**。
3. **规格化**：左规 (尾数溢出或消去) 或右规 (尾数绝对值小于 1/2)。
4. **舍入处理**：0 舍 1 入法、恒置 1 法。
5. **溢出判断**：由阶码决定。
